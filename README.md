## ğŸ“¦ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/nishanttomar21/gppl_website.git
    cd gppl_website
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```
